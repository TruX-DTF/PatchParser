                <h2 class="toggle-title">Description</h2>
               </div>
               <div class="mod-content">
                <div id="description-val" class="field-ignore-highlight"> 
                 <div class="user-content-block"> 
                  <p>I have a table<br> PEOPLE(SEQ_ID INT NOT NULL PRIMARY KEY, PICTURE BLOB).</p> 
                  <p>A row is inserted; both values are not NULL.</p> 
                  <p>From inside a JDBC program, I select the Blob for update.<br> I then get the Blob output stream with a call to<br> Blob.setBinaryStream(long)<br> To this stream I write several times with<br> OutputStream.write(byte[], int, int)<br> I close the stream, update the selected row with the new Blob and commit.</p> 
                  <p>The new value of the Blob now is exactly the value of the last content of the byte[],<br> and it is like the previous calls to write() have never taken place, or as if the file pointer<br> of the output stream has been reset between the calls.</p> 
                  <p>A sample program follows; the size of the input file "picture.jpg" is 23237, the length<br> of the Blob after the program has run is 23237 % 1024 = 709</p> 
                  <p>------------ sample program -------------<br> import java.sql.*;</p> 
                  <p>class TestApp {<br> private TestApp() {}</p> 
                  <p> public static void main(String[] args)<br> throws ClassNotFoundException, SQLException, java.io.IOException {<br> // try to load JDBC driver<br> Class.forName("com.ibm.db2.jcc.DB2Driver");</p> 
                  <p> // open the input file<br> java.io.InputStream instream = new java.io.FileInputStream("picture.jpg");</p> 
                  <p> // login to database<br> Connection conn = DriverManager.getConnection(<br> "jdbc:derby:net://dbtuxe/testdb", "laurenz", "apassword");<br> conn.setAutoCommit(false);</p> 
                  <p> // select Blob for update<br> PreparedStatement stmt = conn.prepareStatement(<br> "SELECT PICTURE FROM PEOPLE WHERE SEQ_ID=? FOR UPDATE OF PICTURE");<br> stmt.setInt(1, 1);<br> ResultSet rs = stmt.executeQuery();</p> 
                  <p> // get Blob output stream<br> rs.next();<br> Blob blob = rs.getBlob(1);<br> java.io.OutputStream outstream = blob.setBinaryStream(1l);</p> 
                  <p> // copy the input file to the Blob in chunks of 1K<br> byte[] buf = new byte<span class="error">[1024]</span>;<br> int count;<br> while (-1 != (count = instream.read(buf))) </p> { outstream.write(buf, 0, count); System.out.println("Written " + count + " bytes to Blob"); } 
                  <p> // close streams<br> instream.close();<br> outstream.close();</p> 
                  <p> // update Blob with new value<br> String cursor = rs.getCursorName();<br> PreparedStatement stmt2 = conn.prepareStatement(<br> "UPDATE PEOPLE SET PICTURE=? WHERE CURRENT OF " + cursor);<br> stmt2.setBlob(1, blob);<br> stmt2.executeUpdate();</p> 
                  <p> // clean up<br> stmt2.close();<br> stmt.close();<br> conn.commit();<br> conn.close();<br> }<br> }</p> 
                 </div> 
                </div>
