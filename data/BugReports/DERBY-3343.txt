                <h2 class="toggle-title">Description</h2>
               </div>
               <div class="mod-content">
                <div id="description-val" class="field-ignore-highlight"> 
                 <div class="user-content-block"> 
                  <p>The following series of statements fails:</p> 
                  <p>j&gt; connect 'jdbc:derby:test;create=true';<br> ij&gt; create table t (id int primary key generated always as identity);<br> 0 rows inserted/updated/deleted<br> ij&gt; prepare p as 'insert into t(id) values (default)';<br> ij&gt; execute p;<br> 1 row inserted/updated/deleted<br> ij&gt; execute p;<br> ERROR 23505: The statement was aborted because it would have caused a duplicate key value in a unique or primary key constraint or unique index identified by 'SQL080123140906700' defined on 'T'.</p> 
                  <p>There is a more detailed discussion about it here: <a href="http://www.nabble.com/Generate-Always-and-SQLIntegrityConstraintViolationException-td15012038.html#a15018054" class="external-link" rel="nofollow">http://www.nabble.com/Generate-Always-and-SQLIntegrityConstraintViolationException-td15012038.html#a15018054</a></p> 
                 </div>
