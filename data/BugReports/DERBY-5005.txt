                <h2 class="toggle-title">Description</h2>
               </div>
               <div class="mod-content">
                <div id="description-val" class="field-ignore-highlight"> 
                 <div class="user-content-block"> 
                  <p>I have a strange issue that can be reproduced easily with the following objects in schema "test":</p> 
                  <p>create table a (a integer);<br> insert into a (a) values(1);<br> create view v as select * from a;</p> 
                  <p>This works:<br> select test.a.a from test.a where test.a.a &lt;&gt; 2 order by test.a.a asc;</p> 
                  <p>This doesn't work:<br> select test.v.a from test.v where test.v.a &lt;&gt; 2 order by test.v.a asc;</p> 
                  <p>But this does:<br> select test.v.a from test.v where test.v.a &lt;&gt; 2 order by v.a asc;</p> 
                  <p>This is the error I get:<br> Error: 'TEST.V' is not an exposed table name in the scope in which it appears.<br> SQLState: 42X10<br> ErrorCode: -1</p> 
                  <p>I've tried quite a few SELECT clauses, and I think the ORDER BY clause is the only one having this issue.</p> 
                 </div> 
                </div>
