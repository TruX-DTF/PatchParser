                <h2 class="toggle-title">Description</h2>
               </div>
               <div class="mod-content">
                <div id="description-val" class="field-ignore-highlight"> 
                 <div class="user-content-block"> 
                  <p>These constructors use the Java default platform encoding to convert the bytes to a String, this typically leads to bugs on platforms with different encodings.</p> 
                  <p>Replace with code using fixed conversion, or alternative mechanisms. <br> If the call is required its use should be commented as to why it is required.</p> 
                  <p>org.apache.derby.client.am.ClobOutputStream - some existing bug?<br> org.apache.derby.client.am.DateTime - see <a href="https://issues.apache.org/jira/browse/DERBY-877" title="zOS - with DerbyClient getDate(#) fails with IllegalArgumentException - unsupported date format - resultset.java" class="issue-link" data-issue-key="DERBY-877"><del>DERBY-877</del></a><br> org.apache.derby.client.am.sqlca</p> 
                  <p>I generated this list using the Java search in eclipse for references to the constructors<br> String(byte[])<br> String(byte[],int,int)</p> 
                 </div>
