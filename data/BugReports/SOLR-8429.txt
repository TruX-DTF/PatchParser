               <div class="mod-content">
                <div id="description-val" class="field-ignore-highlight"> 
                 <div class="user-content-block"> 
                  <p>If authentication is setup with BasicAuthPlugin, it let's all requests go through if no credentials are passed. This was done to have minimal impact for users who only wishes to protect a few end points (say , collection admin and core admin only)</p> 
                  <p>We can add a flag to <tt>BasicAuthPlugin</tt> to allow only authenticated requests to go in </p> 
                  <p>the users can create the first security.json with that flag</p> 
                  <div class="code panel" style="border-width: 1px;">
                   <div class="codeContent panelContent"> 
                    <pre class="code-java">server/scripts/cloud-scripts/zkcli.sh -z localhost:9983 -cmd put /security.json '{<span class="code-quote">"authentication"</span>: {<span class="code-quote">"class"</span>: <span class="code-quote">"solr.BasicAuthPlugin"</span>, 
<span class="code-quote">"blockUnknown"</span>: <span class="code-keyword">true</span>,
<span class="code-quote">"credentials"</span>: {<span class="code-quote">"solr"</span>: <span class="code-quote">"orwp2Ghgj39lmnrZOTm7Qtre1VqHFDfwAEzr0ApbN3Y= Ju5osoAqOX8iafhWpPP01E5P+sg8tK8tHON7rCYZRRw="</span>}}}'
</pre> 
                   </div>
                  </div> 
                  <p>or add the flag later<br> using the command</p> 
                  <div class="code panel" style="border-width: 1px;">
                   <div class="codeContent panelContent"> 
                    <pre class="code-java">curl  http:<span class="code-comment">//localhost:8983/solr/admin/authentication -H <span class="code-quote">'Content-type:application/json'</span> -d  '{ 
</span>{set-property:{blockUnknown:<span class="code-keyword">true</span>}
}'
</pre> 
                   </div>
                  </div>
