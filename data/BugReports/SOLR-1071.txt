                <h2 class="toggle-title">Description</h2>
               </div>
               <div class="mod-content">
                <div id="description-val" class="field-ignore-highlight"> 
                 <div class="user-content-block"> 
                  <p>When: wt=json &amp; spellcheck.extendedResults=true &amp; spellcheck.count &gt; 1, the suggestions are returned in the following format:</p> 
                  <p>"suggestions":[<br> "amsterdm",{<br> "numFound":5,<br> "startOffset":0,<br> "endOffset":8,<br> "origFreq":0,<br> "suggestion":</p> { "frequency":8498, "word":"amsterdam"} 
                  <p>,<br> "suggestion":</p> { "frequency":1, "word":"amsterd"} 
                  <p>,<br> "suggestion":</p> { "frequency":8, "word":"amsterdams"} 
                  <p>,<br> "suggestion":</p> { "frequency":1, "word":"amstedam"} 
                  <p>,<br> "suggestion":{<br> "frequency":22,<br> "word":"amsterdamse"}},<br> "beak",{<br> "numFound":5,<br> "startOffset":9,<br> "endOffset":13,<br> "origFreq":0,<br> "suggestion":</p> { "frequency":379, "word":"beek"} 
                  <p>,<br> "suggestion":</p> { "frequency":26, "word":"beau"} 
                  <p>,<br> "suggestion":</p> { "frequency":26, "word":"baak"} 
                  <p>,<br> "suggestion":</p> { "frequency":15, "word":"teak"} 
                  <p>,<br> "suggestion":{<br> "frequency":11,<br> "word":"beuk"}},<br> "correctlySpelled",false,<br> "collation","amsterdam beek"]}}</p> 
                  <p>This is an invalid json as each term is associated with a JSON object which holds multiple "suggestion" attributes. When working with a JSON library only the last "suggestion" attribute is picked up.</p> 
                 </div> 
                </div> 
               </div>
