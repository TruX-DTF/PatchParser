               <div class="mod-content">
                <div id="description-val" class="field-ignore-highlight"> 
                 <div class="user-content-block"> 
                  <p>The Apache Derby Core and Apache Derby UI plug-ins do not contain the appropriate Apache Software Foundation license files in their zip distributions, derby_core_plugin_10.1.3.zip, derby_ui_plugin_1.1.0.zip.</p> 
                  <p>I'm not sure exactly which files are required, LICENSE, COPYRIGHT and NOTICE<img class="emoticon" src="/jira/images/icons/emoticons/help_16.png" height="16" width="16" align="absmiddle" alt="" border="0">, but the zip files<br> should contain some or all of these files.</p>
