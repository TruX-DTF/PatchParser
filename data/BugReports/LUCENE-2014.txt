                <h2 class="toggle-title">Description</h2>
               </div>
               <div class="mod-content">
                <div id="description-val" class="field-ignore-highlight"> 
                 <div class="user-content-block"> 
                  <p>If i use LUCENE_VERSION &gt;= 2.9 with smart chinese analyzer, it will crash indexwriter with any reasonable amount of chinese text.</p> 
                  <p>its especially annoying because it happens in 2.9.1 RC as well.</p> 
                  <p>this is because the position increments for tokens after stopwords are bogus:</p> 
                  <p>Here's an example (from test case), where the position increment should be 2, but is instead 91975314!</p> 
                  <div class="code panel" style="border-width: 1px;">
                   <div class="codeContent panelContent"> 
                    <pre class="code-java">  <span class="code-keyword">public</span> void testChineseStopWords2() <span class="code-keyword">throws</span> Exception {
    Analyzer ca = <span class="code-keyword">new</span> SmartChineseAnalyzer(Version.LUCENE_CURRENT); <span class="code-comment">/* will load stopwords */</span>
    <span class="code-object">String</span> sentence = <span class="code-quote">"Title:San"</span>; <span class="code-comment">// : is a stopword
</span>    <span class="code-object">String</span> result[] = { <span class="code-quote">"titl"</span>, <span class="code-quote">"san"</span>};
    <span class="code-object">int</span> startOffsets[] = { 0, 6 };
    <span class="code-object">int</span> endOffsets[] = { 5, 9 };
    <span class="code-object">int</span> posIncr[] = { 1, 2 };
    assertAnalyzesTo(ca, sentence, result, startOffsets, endOffsets, posIncr);
  }
</pre> 
                   </div>
                  </div> 
                  <p>junit.framework.AssertionFailedError: posIncrement 1 expected:&lt;2&gt; but was:&lt;91975314&gt;<br> at junit.framework.Assert.fail(Assert.java:47)<br> at junit.framework.Assert.failNotEquals(Assert.java:280)<br> at junit.framework.Assert.assertEquals(Assert.java:64)<br> at junit.framework.Assert.assertEquals(Assert.java:198)<br> at org.apache.lucene.analysis.BaseTokenStreamTestCase.assertTokenStreamContents(BaseTokenStreamTestCase.java:83)<br> ...</p> 
                 </div> 
                </div>
