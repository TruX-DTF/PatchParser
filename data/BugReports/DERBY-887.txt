                  <p>I have a table with 2 columns one is BIGINT which is Generated always starting with 1 and increment by1 and the other one is timestamp</p> 
                  <p> ij&gt; select id,date from inbox where id = 0&lt;3;<br> ID |DATE<br> -----------------------------------------------<br> 25 |2006-01-26 14:35:46.584<br> 34 |2006-01-26 14:36:16.588<br> 21 |2006-01-26 14:34:46.455<br> 22 |2006-01-26 14:34:47.176<br> 27 |2006-01-26 14:35:47.054<br> 24 |2006-01-26 14:35:16.58<br> 28 |2006-01-26 14:35:47.305<br> 35 |2006-01-26 14:36:18.771<br> 31 |2006-01-26 14:35:48.496<br> 32 |2006-01-26 14:35:48.887<br> 33 |2006-01-26 14:35:49.308</p> 
                  <p>11 rows selected<br> ij&gt; select id,date from inbox where id = true;<br> ID |DATE<br> -----------------------------------------------<br> 21 |2006-01-26 14:34:46.455</p> 
                  <p>1 row selected</p> 
                  <p>Both queries should return same number of rows</p> 
                  <p>Also If I delete the first row from the table then select id,date from inbox where id = true; returns the second row<br> ij&gt; delete from inbox where id=21;</p> 
                  <p>1 row inserted/updated/deleted</p> 
                  <p>Then again run<br> ij&gt; select id ,date from inbox where id=true;<br> ID |DATE<br> -----------------------------------------------<br> 22 |2006-01-26 14:35:16.58</p> 
                  <p>Which indicates that the select returns only the first row and then is getting closed</p>
