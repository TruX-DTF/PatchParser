                <div id="description-val" class="field-ignore-highlight"> 
                 <div class="user-content-block"> 
                  <p>While reviewing <tt>SolrIndexSearcher</tt>, I started to correct a thing here and there, and eventually it led to these changes:</p> 
                  <ul> 
                   <li>Moving <tt>QueryCommand</tt> and <tt>QueryResult</tt> to their own classes.</li> 
                   <li>Moving FilterImpl into a private static class (was package-private and defined in the same .java file, but separate class).</li> 
                   <li>Some code formatting, imports organizing and minor log changes.</li> 
                   <li>Removed fieldNames (handled the TODO in the code)</li> 
                   <li>Got rid of usage of deprecated classes such as <tt>LegacyNumericUtils</tt> and <tt>Legacy-*-Field</tt>.</li> 
                  </ul> 
                  <p>I wish we'd cut down the size of this file much more (it's 2500 lines now), but I've decided to stop here so that the patch is manageable. I would like to explore further refactorings afterwards, e.g. extracting cache management code to an outer class (but keep <tt>SolrIndexSearcher</tt>'s API the same, if possible).</p> 
                  <p>If you have additional ideas of more cleanups / simplifications, I'd be glad to do them.</p>
