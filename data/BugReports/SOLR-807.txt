               <div class="mod-content">
                <div id="description-val" class="field-ignore-highlight"> 
                 <div class="user-content-block"> 
                  <p>I'm using UUID via Solrj in my project. When I use javabin (default), I got:</p> 
                  <p><b>java.util.UUID:</b> 391e3214-4f8e-4abd-aa6b-4f12be79534f</p> 
                  <p>as the uuid value. But if I use xml, I got:</p> 
                  <p>391e3214-4f8e-4abd-aa6b-4f12be79534f</p> 
                  <p>I think the both of them should return same string.</p> 
                  <p>program for reproducing the problem:</p> 
                  <div class="code panel" style="border-width: 1px;">
                   <div class="codeContent panelContent"> 
                    <pre class="code-java">  <span class="code-keyword">public</span> <span class="code-keyword">static</span> void main(<span class="code-object">String</span>[] args) <span class="code-keyword">throws</span> Exception {
    CommonsHttpSolrServer server = <span class="code-keyword">new</span> CommonsHttpSolrServer( <span class="code-quote">"http:<span class="code-comment">//localhost:8983/solr"</span> );
</span>    SolrQuery query = <span class="code-keyword">new</span> SolrQuery().setQuery( <span class="code-quote">"*:*"</span> );
    <span class="code-comment">//server.setParser( <span class="code-keyword">new</span> XMLResponseParser() );   // uncomment <span class="code-keyword">for</span> wt=xml
</span>    <span class="code-object">System</span>.out.println( <span class="code-quote">"===== "</span> + server.getParser().getClass().getSimpleName() + <span class="code-quote">" ====="</span> );
    QueryResponse rsp = server.query( query );
    SolrDocumentList docs = rsp.getResults();
    <span class="code-keyword">for</span>( SolrDocument doc : docs ){
      <span class="code-object">Object</span> id = doc.getFieldValue( <span class="code-quote">"id"</span> );
      <span class="code-object">System</span>.out.println( <span class="code-quote">"type = "</span> + id.getClass().getName() + <span class="code-quote">", id = "</span> + id );
      <span class="code-object">Object</span> timestamp = doc.getFieldValue( <span class="code-quote">"timestamp"</span> );
      <span class="code-object">System</span>.out.println( <span class="code-quote">"type = "</span> + timestamp.getClass().getName() + <span class="code-quote">", timestamp = "</span> + timestamp );
    }
  }
</pre> 
                   </div>
                  </div> 
                  <p>result for wt=javabin</p> 
                  <div class="code panel" style="border-width: 1px;">
                   <div class="codeHeader panelHeader" style="border-bottom-width: 1px;">
                    <b>javabin</b>
                   </div>
                   <div class="codeContent panelContent"> 
                    <pre class="code-java">===== BinaryResponseParser =====
type = java.lang.<span class="code-object">String</span>, id = java.util.UUID:391e3214-4f8e-4abd-aa6b-4f12be79534f

</pre>
