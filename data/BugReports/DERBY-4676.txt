                 <div class="user-content-block"> 
                  <p>Running into a NullPointerException error in the Apache Derby database over multiple versions of the derby jars. From testing, this issue intermittently occurs during moderate load test scenarios, but has never occurred in production. This is using Derby as embedded and always occurs on the same statement as shown below and in the attachment. Following the error, hibernate throws an exception which results in the code attempting to rollback the transaction. The rollback fails as the NullPointerException appears to kill the connection.</p>
