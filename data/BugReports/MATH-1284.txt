               <div class="mod-content">
                <div id="description-val" class="field-ignore-highlight"> 
                 <div class="user-content-block"> 
                  <p>The class hierarchy for geometry claims that Vector is-a Point: <a href="https://commons.apache.org/proper/commons-math/apidocs/org/apache/commons/math3/geometry/Point.html" class="external-link" rel="nofollow">https://commons.apache.org/proper/commons-math/apidocs/org/apache/commons/math3/geometry/Point.html</a></p> 
                  <p>This is mathematically incorrect, see e.g. <a href="http://math.stackexchange.com/a/645827" class="external-link" rel="nofollow">http://math.stackexchange.com/a/645827</a></p> 
                  <p>Just because they share the same numerical representation, Point and Vector shouldn't be crammed into a common class hierarchy.</p> 
                 </div>
