               <div class="mod-content">
                <div id="description-val" class="field-ignore-highlight"> 
                 <div class="user-content-block"> 
                  <p>I have seen a few bugs around this recently: of course its a bug in application code but a JVM crash is not good. </p> 
                  <p>I think we should see if we can prevent the crashes better than the current weak map, e.g. make it a safer option.</p> 
                  <p>I made an ugly prototype here: <a href="https://github.com/apache/lucene-solr/compare/master...rmuir:ace?expand=1" class="external-link" rel="nofollow">https://github.com/apache/lucene-solr/compare/master...rmuir:ace?expand=1</a></p> 
                  <p>It has a test that crashes the JVM without the patch but passes with.</p>
