                <h2 class="toggle-title">Description</h2>
               </div>
               <div class="mod-content">
                <div id="description-val" class="field-ignore-highlight"> 
                 <div class="user-content-block"> 
                  <p>The newly introduced MoreLikeThis Query Parser (<a href="https://issues.apache.org/jira/browse/SOLR-6248" title="MoreLikeThis Query Parser" class="issue-link" data-issue-key="SOLR-6248"><del>SOLR-6248</del></a>) does not return any results when supplied with multiple fields in the <tt>qf</tt> parameter.</p> 
                  <p>To reproduce within the techproducts example, compare:</p> 
                  <div class="code panel" style="border-width: 1px;">
                   <div class="codeContent panelContent"> 
                    <pre class="code-java">curl <span class="code-quote">'http:<span class="code-comment">//localhost:8983/solr/techproducts/select?q=%7B!mlt+qf=name%7DMA147LL/A'</span>
</span>curl <span class="code-quote">'http:<span class="code-comment">//localhost:8983/solr/techproducts/select?q=%7B!mlt+qf=features%7DMA147LL/A'</span>
</span>curl <span class="code-quote">'http:<span class="code-comment">//localhost:8983/solr/techproducts/select?q=%7B!mlt+qf=name,features%7DMA147LL/A'</span></span>
</pre> 
                   </div>
                  </div> 
                  <p>The first two queries return 8 and 5 results, respectively. The third query doesn't return any results (not even the matched document).</p> 
                  <p>In contrast, the MoreLikeThis Handler works as expected (accounting for the default <tt>mintf</tt> and <tt>mindf</tt> values in SimpleMLTQParser):</p> 
                  <div class="code panel" style="border-width: 1px;">
                   <div class="codeContent panelContent"> 
                    <pre class="code-java">curl <span class="code-quote">'http:<span class="code-comment">//localhost:8983/solr/techproducts/mlt?q=id:MA147LL/A&amp;mlt.fl=name&amp;mlt.mintf=1&amp;mlt.mindf=1'</span>
</span>curl <span class="code-quote">'http:<span class="code-comment">//localhost:8983/solr/techproducts/mlt?q=id:MA147LL/A&amp;mlt.fl=features&amp;mlt.mintf=1&amp;mlt.mindf=1'</span>
</span>curl <span class="code-quote">'http:<span class="code-comment">//localhost:8983/solr/techproducts/mlt?q=id:MA147LL/A&amp;mlt.fl=name,features&amp;mlt.mintf=1&amp;mlt.mindf=1'</span></span>
</pre> 
                   </div>
                  </div> 
                  <p>After adding the following line to <tt>example/techproducts/solr/techproducts/conf/solrconfig.xml</tt>:</p> 
                  <div class="code panel" style="border-width: 1px;">
                   <div class="codeContent panelContent"> 
                    <pre class="code-java">&lt;requestHandler name=<span class="code-quote">"/mlt"</span> class=<span class="code-quote">"solr.MoreLikeThisHandler"</span> /&gt;
</pre> 
                   </div>
                  </div> 
                  <p>The first two queries return 7 and 4 results, respectively (excluding the matched document). The third query returns 7 results, as one would expect.</p> 
                 </div> 
                </div> 
               </div>
