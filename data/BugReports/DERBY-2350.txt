                <h2 class="toggle-title">Description</h2>
               </div>
               <div class="mod-content">
                <div id="description-val" class="field-ignore-highlight"> 
                 <div class="user-content-block"> 
                  <p>A trigger like, where V, V1 and V2 are columns of type XML will thrown an exception when fired.</p> 
                  <p>CREATE TRIGGER AIS AFTER INSERT ON T_MAIN <br> REFERENCING NEW_TABLE AS N<br> FOR EACH STATEMENT <br> INSERT INTO T_ACTION_STATEMENT(A, V1, ID, V2) <br> SELECT 'I', V, ID, V FROM N</p> 
                  <p>ERROR 38000: The exception 'java.sql.SQLException: An attempt was made to get a data value of type 'java.lang.Object' from a data value of type 'XML'.' was thrown while evaluating an expression.</p> 
                  <p>Most likely because triggers are implementing using VTIs and hence JDBC ResultSets and XML is not supported through JDBC yet.</p> 
                  <p>TriggerTest shows this issue, see the comment with the bug number to reproduce.</p> 
                 </div> 
                </div>
