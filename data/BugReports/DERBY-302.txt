               </div>
               <div class="mod-content">
                <div id="description-val" class="field-ignore-highlight"> 
                 <div class="user-content-block"> 
                  <p>I downloaded a Cloudescape Version 10 from here <a href="http://www-106.ibm.com/developerworks/db2/library/techarticle/dm-0408cline/index.html" class="external-link" rel="nofollow">http://www-106.ibm.com/developerworks/db2/library/techarticle/dm-0408cline/index.html</a>. Not sure exactly which version of 10 this is.</p> 
                  <p>Would like to store up to a 10MB XML string in a CLOB. However, I noticed it took over 3 minutes to insert a 500kb string. I am using the PreparedStatement to get around the 37kb limitation encountered in a previous issue. It takes less than 2 seconds to insert a 500kb string into a CLOB in mySQl.</p> 
                  <p>Here the snippet:</p> 
                  <p> FileReader fr = new FileReader ("sample.txt"); <br> BufferedReader br = new BufferedReader(fr);<br> String record = null;<br> try {</p> 
                  <p> while ( (record=br.readLine()) != null ) </p> { bufferStr.append( record ); } 
                  <p> } catch (IOException e) </p> { // // put your error-handling code here // System.out.println("Error reading file"); } 
                  <p> System.out.println("Size of inputStr: "+bufferStr.length() );</p> 
                  <p> PreparedStatement ps = mm.connection.prepareStatement("INSERT into documents VALUES (?,?)" );<br> ps.setInt(1, 13 );<br> StringReader reader1 = new StringReader(bufferStr.toString());<br> ps.setCharacterStream(2, reader1, bufferStr.length());<br> System.out.println("Uploading string....");<br> ps.execute();<br> System.out.println("Done uploading string...");<br> mm.connection.commit();</p> 
                  <p>Thanks for your the help,</p> 
                  <p>Glenn O.</p> 
                 </div>
